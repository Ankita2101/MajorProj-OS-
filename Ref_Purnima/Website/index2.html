ABOUT SOFTWARE

This is an educational software, Which is proposed to demonstrate and verify basic Operating System Concepts such as Process Scheduling, Process Synchronization, Deadlock and Page Replacement Algorithms.

This software transforms the theory of above topics into graphical view. As well as anyone can test and compare the results for each simulation and can the text or excel file.

To use this software one should have the basic knowledge of Operating System's concepts.To know the basics to productively use this software one can find the user manual in this website.
